Difference between let, const, and var?

var is function-scoped and allows redeclaration, while let and const are block-scoped. let allows reassignment, but not redeclaration in the same scope, whereas const is a constant and cannot be reassigned.

What is a closure?

A closure lets an inner function access variables from an outer function even after the outer function finishes running. Closures are useful for creating private variables and maintaining state across function calls.

Difference between == and ===?

== (loose equality) allows type conversion during comparison, while === (strict equality) requires both type and value to match exactly, making it more reliable for precise comparisons.

What is this in JavaScript?

this refers to the object calling the function. In a method, it’s the object itself; in a regular function, it’s usually global or undefined in strict mode; and in an arrow function, it takes this from its surrounding scope.
